# Security Policy

God's Hands is security-sensitive software. Please do not publish exploit details in public issues.

## Reporting a Vulnerability

If you find a vulnerability, report it privately to the repository owner. Include:

- Affected version or commit.
- Clear reproduction steps.
- Expected impact.
- Any suggested mitigation.

## Scope Notes

Simple mode locks under five minutes are convenience locks and do not provide cryptographic time-lock guarantees. NIST mode depends on live access to the NIST Randomness Beacon API.
